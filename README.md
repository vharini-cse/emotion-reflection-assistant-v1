# emotion-reflection-assistant-v1
 Emotion Reflection Assistant built using Python, Streamlit, and NLTK Sentiment Analysis.
Emotion Reflection Assistant Version 1
Emotion Reflection Assistant is a web-based mindfulness application developed using Python, Streamlit, and NLTK Sentiment Analysis. The application helps users reflect on their emotional well-being by identifying emotions from user input, analyzing sentiment, providing personalized reflection questions, recommending yoga breathing exercises, and generating a wellness reflection score.
Features:
• Emotion Detection
• NLTK Sentiment Analysis
• Personalized Reflection Questions
• Yoga Breathing Recommendations
• Positive Daily Quotes
• Wellness Reflection Score
• Interactive Streamlit Web Interface
Tech Stack:
• Python
• Streamlit
• NLTK
This project was developed as part of International Yoga Day to promote self-awareness, mindfulness, and emotional wellness through technology.
